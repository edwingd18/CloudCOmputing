Run Application
================

1. Start and SSH into the Vagrant VM
------------------------------------

```bash
vagrant up
vagrant ssh servidorWeb
```

2. Start the microservices and frontend
---------------------------------------

```bash
./iniciar_microservicios.sh
```

3. Access Consul UI
-------------------

Open the following URL in your browser:

```
http://192.168.50.4:8500
```
